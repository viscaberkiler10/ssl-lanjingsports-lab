# ssl-lanjingsports-lab

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面，主要目的是整理和存放与特定主题相关的静态资源。所有页面均为自包含的 HTML 文件，不依赖外部域名或特定网站，仅作为实验室环境下的资料存储与展示。

## 目录结构

```
ssl-lanjingsports-lab/
├── docs/          # 存放生成的 HTML 页面及相关静态资源
├── assets/        # 公共资源文件（如样式、脚本、图片等）
├── src/           # 页面源文件或原始数据（可选）
└── README.md      # 本文件
```

- `docs/`：该目录下的 HTML 文件可通过 GitHub Pages 直接访问，用于呈现归档内容。
- `assets/`：存放多个页面共享的 CSS、JavaScript 或图片文件，方便统一管理。
- `src/`：若包含原始素材或构建脚本，可放置于此。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，每个页面对应一个特定的主题或实验内容。页面之间无强关联，可单独查看或部署。所有页面遵循基本的 Web 标准，力求简洁清晰，便于直接打开阅读。
